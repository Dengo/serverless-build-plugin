# SERVERLESS BUILD PLUGIN
This plugin is intended to add flexibility to the serverless build process for serverless 1.0.

### FEATURES
- Build files export:
    - Functions (which should return any of the below)
    - Webpack configs
    - File strings & buffers
    - File streams
